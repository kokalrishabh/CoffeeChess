# CoffeeChess
This is an application that tests you on your opening knowledge. Simply pick a PGN file to practice from and the color you want to play as and the computer will test you on random lines.

## Context

In chess, there is a concept called opening theory. The first moves in many openings have been explored, and theory dictates what the best move is in a given opening position. Knowing your theory can help you get good or even winning positions out of the opening; on the other hand, not knowing your theory can result in you falling into traps or playing suboptimal moves that land you in worse or losing positions.

A set of moves played in response to one another is called a line. Collections of lines can be saved as plaintext files called PGNs (Portable Game Notation files). These PGNs are used by many chess softwares, including this one.

## The Application

Simply select a file that you'd like to test yourself on, choose the color with which you'd like to practice, change any options you'd like under "More", and start practicing. The computer will make moves against you, selecting from the given PGN file at random, giving you the opportunity to drill your lines well.

The arrows in the bottom left corner allow you to go back to moves previously played (two at at time, so as to allow you to play from any previous position). The flip button allows you to see the board from the other perspective. The reroll button allows you to have the computer select a different move if for whatever reason you would like to test yourself on a different line. The "I don't know" button tells you all moves listed in the given PGN in the displayed position.
